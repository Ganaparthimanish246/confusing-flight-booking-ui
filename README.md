# confusing-flight-booking-ui

## Explanation
This project was created for the Worst UI Challenge, where the goal is to intentionally violate UI/UX principles. The design includes misleading navigation, random button placement, unclear actions, excessive colors, and confusing booking steps to demonstrate how poor design impacts usability.

## Features
- Over 20 colorful, overlapping buttons with unclear labels.
- Misleading Navigation Bar where icons don't match the labels or the destination.
- Confusing Flight Search inputs with a destination dropdown opening behind other elements.
- Randomly colored prices in the Flight Results page and indistinguishable airline labels.
- "Cancel" confirms booking, "Confirm" resets the form.
- Multiple fake payment options and an infinite loading bar.

## Getting Started

To run this project:
1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter run -d chrome` or `flutter run -d windows`.

Enjoy the terrible UX!
